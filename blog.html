Important FAQ

What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a tree-like structure that represents a webpage's content, allowing JavaScript to access and manipulate HTML elements, styles, and structure dynamically.

How do you select an element from the DOM?

I can select an element from the DOM by - getElementById(id), getElementsByClassName(className), getElementsByTagName(tagName), querySelector(selector), querySelectorAll(selector),

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in JavaScript is a method that enables the management of events on a parent element instead of on individual child elements. It is especially beneficial when multiple child elements need to react to the same event, such as clicks, or when child elements are added to or removed from the DOM dynamically.

How do you manipulate an element's attributes and styles using the DOM?

I can use getAttribute(), setAttribute(), and removeAttribute() to manipulate attributes. for example: element.getAttribute("attributeName"), element.setAttribute("attributeName", "value") , element.removeAttribute("attributeName"). On the other hand, I can use the style property and classList to manipulate an element's styles. for example: Inline Styles: element.style.propertyName = "value", Add: element.classList.add("className") Remove: element.classList.remove("className")
